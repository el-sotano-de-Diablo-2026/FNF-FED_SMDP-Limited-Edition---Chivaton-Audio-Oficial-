# FNF-FED_SMDP-Limited-Edition---Chivaton-Audio-Oficial-
Esta es la canción que nunca se utilizó en el mod de Friday Night Funkin "FED_SMDP Limited Edition"


Si está leyendo esto, es importante que se mantenga el incógnito de este material de suma importancia. El 12 de abril de 2026, entre Diablo y Francisco El Desastre (FED_SMDP) crearon en FL Studio Mobile una de las canciones que formaría parte del mod de Friday Night Funkin "FED_SMDP: Limited Edition", pero en la finalización de dicho mod, la canción del nivel "Chivaton" sería removida para que nadie supiera nada de su existencia. Nunca se publicó ninguna información sobre su presencia en el mod, por lo que nadie sabe nada de este material.
